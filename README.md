This is my most recent project. It is a Temperture sensor based fan controller. 
It works using the inputs from 3x DS18B20 Temperature sensors, Arduino Nano microcontroller, and an onboard relay to switch the fan on or off, based on the thresholds set by the programer.
The reason why 3 sensors were chosen was to ensure uniform temperature equalization throughout the different levels of a particular room. 

I have included the Kicad files in the fan_controller.zip file, along with some screenshots of my design. 
Hope you like it :)
